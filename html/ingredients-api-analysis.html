🌿 D. Tab 2 — 食材特性 detail/components/Nutrients.vue
ℹ️ 此 Tab 不發出額外 API 請求。
• 產季月份：純前端 checkbox 選取，儲存時隨 PUT /pdm/ingredient/update 的 season 欄位一起送出。
• 相生相剋清單：資料由詳細頁 GET /pdm/ingredient-compat/getList（B7）載入，在 Tab 內直接操作陣列，儲存時隨 B9 送出。
